# Web fonts (WOFF2)

Place **four** Grift files here (replace any placeholders):

- `Grift-Regular.woff2` — 400  
- `Grift-Medium.woff2` — 500  
- `Grift-SemiBold.woff2` — 600  
- `Grift-Bold.woff2` — 700  

They are loaded via `next/font/local` in `lib/fonts.ts`.

**Note:** Fresh clones may ship with SIL-licensed Inter-derived placeholders under these names so `next build` succeeds until you add real Grift files from your font license.
